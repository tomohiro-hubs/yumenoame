# 斑鳩の米飴 - ゆめの飴 コーポレートサイト

## 🌾 プロジェクト概要

奈良・斑鳩町の米と麦芽だけで作る自然甘味料「斑鳩の米飴」のコーポレートサイトです。
「令和のモダン禅」をテーマに、Awwwards級のビジュアルインパクトと情報の可読性を両立させた、芸術的なWEBサイトとして設計されています。

### プロジェクトの目標
- 家族の健康を気づかう30-40代の主婦層をメインターゲットとしたECサイトへの誘導
- ブランドストーリーと製品価値の効果的な伝達
- 視覚的なインパクトと情報の可読性の両立
